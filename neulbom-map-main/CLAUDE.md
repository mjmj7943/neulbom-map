# 프로젝트 개요

화성오산 늘봄자원지도 (neulbom-map)  
화성·오산 지역의 늘봄학교 관련 지역 자원을 시각화하는 정적 웹 서비스.

# 기술 스택

- 순수 HTML / CSS / JavaScript (프레임워크 없음)
- 지도: Leaflet.js
- 데이터: Google Sheets (공개 API 또는 CSV 내보내기 방식)
- 별도 빌드 과정 없이 브라우저에서 바로 실행되는 구조

# 작업 지침

- 모든 대화는 한국어로 진행할 것.
- 코드 주석도 한국어로 작성할 것.
- 전체 구조 참조가 필요한 경우 PROJECT_ANALYSIS.md를 참고하고 업데이트할 것.
- 프레임워크나 번들러를 도입하지 말 것. 순수 HTML/CSS/JS 구조를 유지할 것.
- 외부 라이브러리는 CDN으로 불러오는 방식을 사용할 것.
- 코드 변경 시 변경 이유와 내용을 간단히 설명할 것.
